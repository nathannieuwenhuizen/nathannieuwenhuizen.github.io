Double Dice

Info

Platform: Web

Team: 1 Developer

Date: November 2016

Play: 3D version 2D version

Github: click here

Description

As part of a test assignment for an internship at Mindgame, I made this small dice game.

The rules are simple: try to throw the dices with the same number of eyes to each other. If you succeed, you get live up, if not then one down.

p>

Try to get as many turns as you can before your lives are up.

My role

  • Programmed the UI
  • Animated the dices and cup with CSS.
  • Designed the appearance of the game.

  • Simple gamle game
